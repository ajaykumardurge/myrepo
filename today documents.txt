7.  rmdir

Delete a directory , the rm command is used to delete directories and the contents within them. 
If you only want to delete the directory — as an alternative to rmdir — use rm -r.

Note: Be very careful with this command and double-check which directory you are in. This will delete everything and there is no undo.

18. kill command

If you have an unresponsive program, you can terminate it manually by using the kill command. 
It will send a certain signal to the misbehaving app and instructs the app to terminate itself.

20. ping command

Use the ping command to check your connectivity status to a server.
For example, by simply entering ping google.com, the command will check whether you’re able to connect to Google and also measure the response time.
